¿Cuánto sabes de Desarrollo Web?

¿Qué es HTML?

Es un lenguaje fundamental de etiquetado para el desarrollo de páginas web, basado en el uso de etiquetas (tags) intercaladas dentro de documentos de texto plano. Estas etiquetas son interpretadas por los navegadores (browsers) para estructurar el contenido y definir su presentación visual al usuario. Los archivos HTML, que pueden crearse fácilmente con cualquier editor de texto, se guardan comúnmente con la extensión .htm o .html.

¿Qué es CSS?

Es un lenguaje formal crucial de estilo para definir la presentación estética de documentos estructurados, especialmente aquellos escritos en HTML. Su principal objetivo es separar la estructura y el contenido del documento de su diseño visual, lo que otorga un control significativamente mayor sobre los atributos de presentación y convierte al HTML en un lenguaje más versátil, liviano y fácil de mantener.

¿Qué es JavaScript?

Este lenguaje de programación se caracteriza por ser interpretado, lo que significa que su código se ejecuta línea por línea sin una compilación previa; es multiparadigma, permitiendo la combinación flexible de diferentes estilos de programación; y posee tipado dinámico, lo cual implica que las variables pueden cambiar el tipo de dato que almacenan durante el tiempo de ejecución del programa.
